# Task

**Running an application requires entering database settings in "application.settings".** 

**Enter 'mvn test' in terminal to runn all unit tests** 

**Features:**
 - REST API: 
   - api/flights/
   - api/flights/{id}
   - api/tourists
   - api/tourists/{id}
   - api/reservation
   - api/reservation/{id}
   
   **Technologies:**
 - Spring Boot,
 - Hibernate,
 - MySQL,
 - REST
 - jUnit5 and Mockito
